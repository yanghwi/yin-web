# Yin

새로운 프로젝트를 시작하기 위한 저장소입니다.

## 프로젝트 개요

Next.js 14 기반의 미니멀 1열 블로그입니다. Markdown 파일로 작성된 글을
정적 라우트(`/YYYY/MM/DD/slug/`)로 노출하며, 상단 전역 네비게이션과 이전·다음
글 이동 링크를 제공합니다.

## 개발 환경 준비

1. 의존성 설치

   ```bash
   npm install
   ```

2. 개발 서버 실행

   ```bash
   npm run dev
   ```
